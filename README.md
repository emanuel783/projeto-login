# projeto-login

### descrição 

Projeto criado na aula da disciplinade programação Front-End, na Unicesumar, campus de Londrina. O professor [Leonardo Rocha](https://github.comleonardossrocha) introduziu os conceitos de framework, apresentando o Bootstrap e alguns exemplos disponíveis nesse framework, e git, realizando o passo-a-passo de configuração do ambiente de desenvolvimento. 

### Ferramentas utilizadas 

- HTML5
- CSS3
- Bootstrap 
- Git
